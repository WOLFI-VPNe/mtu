# mtu

This repository contains a network MTU discovery and performance analysis script.

## Files

- `discover_mtu.sh` - Path MTU discovery and network performance analyzer for IPv4/IPv6.

## Usage

Run the script with a target IP address:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/WOLFI-VPNe/mtu/main/mtu.sh)
```

Or run without arguments and enter the target IP when prompted.
